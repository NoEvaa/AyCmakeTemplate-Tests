# C++ Cmake Template: Tests

A cmake template for tests of C++ library.

更多相关内容请移步[AyCmakeTemplate](https://gitee.com/ayin-tech/ay-cmake-template)。

## 适用场景

- 测试用例

## 快速开始

### 构建

```bash
sh build.sh
sh build.sh Release
ctest --test-dir ./build -O ./build/Testing
```
