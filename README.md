# C++ Cmake Template: Tests

A cmake template for tests of C++ library.

See [AyCmakeTemplate](https://gitee.com/ayin-tech/ay-cmake-template) for more.

## Quick Start

### Building

```bash
sh build.sh
sh build.sh Release
ctest --test-dir ./build -O ./build/Testing
```
